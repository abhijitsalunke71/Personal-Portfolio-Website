# Personal-Portfolio-Website
The website is designed with a clean, modern interface and follows responsive design principles to ensure seamless performance across desktops, tablets, and mobile devices. It includes well-organized sections such as About Me, Skills, Projects, Education, and Contact, making navigation intuitive and user-friendly.

Designed with a focus on simplicity, responsiveness, and performance, this portfolio serves as a central place to present my work and connect with recruiters and collaborators.
